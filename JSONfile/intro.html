To know some JSON 

        
        -Data is in key/value pairs: "key": "value"
        -Data is separated by commas: ,
        -Curly braces {} hold objects.
        -Square brackets [] hold arrays.
        -Keys must be strings, enclosed in double quotes.
        -Values can be a string, number, object, array, true, false, or null.
    
        Uses of JSON:
        
        APIs: 
        JSON is widely used for data exchange in RESTful APIs.
        Configuration files: 
        JSON is often used to store application configuration.
        Database storage: 
        Some databases, like MongoDB, store data in JSON format or a similar structure.
