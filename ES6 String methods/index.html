string methods are case sensitive

        1. startsWith (searchString, position)
        check a strings start with another strings.
        This method checks from the beginning if string is present or not.


        2. endswith (searchString, length)
        check a string ends with another strings.
        This method checks from the end whether string is present or not.

        3. includes (searchString, position)
        check if a string contains another strings.
        This method only checks if string is present or not.

        4. returns (true/false)
